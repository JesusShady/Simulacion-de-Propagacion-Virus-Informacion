## рҹҺҜ ВҝQuГ© es esto?

**BFS Virus Propagation Simulator** es una aplicaciГіn de escritorio que simula visualmente cГіmo un virus se propaga a travГ©s de una red de personas usando:

- рҹ”Қ **BFS (Breadth-First Search)** вҖ” El virus se expande nivel por nivel desde los pacientes cero
- рҹҰ  **Modelo SIR** вҖ” Cada persona pasa por estados: **S**usceptible вҶ’ **I**nfectado вҶ’ **R**ecuperado
- рҹҺ® **Controles interactivos** вҖ” Crea redes, configura parГЎmetros, y observa la propagaciГіn en tiempo real

## рҹ“ҡ Fundamento TeГіrico

### Algoritmo BFS (Breadth-First Search)

```
  BFS usa una COLA (FIFO) para explorar el grafo nivel por nivel:

  Cola: [N0]
  в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
  Iter 0:  Procesar N0 вҶ’ infectar vecinos N1, N3, N5
  Cola: [N1, N3, N5]
  в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
  Iter 1:  Procesar N1 вҶ’ infectar N2, N4
           Procesar N3 вҶ’ infectar N6
           Procesar N5 вҶ’ ya infectado (skip)
  Cola: [N2, N4, N6]
  в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
  Iter 2:  Procesar N2 вҶ’ sin vecinos nuevos
           Procesar N4 вҶ’ infectar N7
           Procesar N6 вҶ’ infectar N8
  Cola: [N7, N8]
  в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
  ...continГәa hasta que la cola estГ© vacГӯa
```

### Modelo SIR

```
  dS/dt = -ОІ * S * I / N
  dI/dt = ОІ * S * I / N - Оі * I
  dR/dt = Оі * I

  Donde:
    ОІ = tasa de contagio (configurable)
    Оі = tasa de recuperaciГіn (1 / tiempo_recuperaciГіn)
    N = poblaciГіn total
```

### Centralidad de Grado

```
  C(v) = grado(v) / (N - 1)

  El nodo con mayor C(v) es el "SГәper Propagador".
  Es el nodo mГЎs peligroso si se infecta.
```
